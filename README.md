:floppy_disk: `install-orocos-kdl`
==================================

This repository contains a GitHub Action to install [Orocos KDL](https://www.orocos.org/kdl.html).

#### How-to-use
```bash
steps:   
  - name: Install Orocos KDL
    uses: lxrobotics/install-orocos-kdl@main
```
